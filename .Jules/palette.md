## 2025-05-14 - Accessibility & Skip Links
**Learning:** Adding a "Skip to content" link and ensuring floating widgets (like ChatWidget) manage focus automatically significantly improves the experience for keyboard and screen reader users. Also, using semantic landmarks like `<nav>` and `<main>` helps users navigate the page structure more easily.
**Action:** Always include skip-to-content links in the root layout and implement focus management for interactive dialogs/modals.
